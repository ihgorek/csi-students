\section*{Заключение}
\addcontentsline{toc}{subsection}{Заключение}

В результате работы методом желаемой ЛАЧХ был спроектирован последовательный регулятор для системы управления, который обеспечивает необходимые показатели качества на выходе замкнутой системы при воздействии на неё сигналом с ограниченной скоростью и ускорением.\par
Метод желаемой логарифмической амплитудно-частотной характеристики удобно применять для синтеза последовательно включённого регулятора. Также, в силу лёгкости составления ЛАЧХ звена, данный метод можно использовать для оценивания порядка полиномов в числителе и знаменателе регулятора, а вместе с тем и сложность его реализации.\par
При разбиении передаточной функции регулятора на элементарные звенья, можно составить рассчитать электрическую схему реализации регулятора.
