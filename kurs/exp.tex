\section{Реализация регулятора}
Для реализации регулятора, необходимо представить передаточную функцию:
\begin{align}
W_p(s) = \frac{(K_v+\sqrt{2})(T_3s + 1)}{s(T_4s + 1)^2(T_ks + 1)}.
\end{align}

В виде произведения типовых динамических звеньев:
\begin{align}
W_p(s) = \frac{T_3s + 1}{T_0s}\cdot\frac{K_v+\sqrt{2}}{T_ks + 1}\cdot\frac{1}{T_4s + 1}\cdot\frac{1}{T_4s + 1}.
\end{align}

Тогда электрическая схема регулятора принет вид, представленный на рисунке \ref{elchem}.
\begin{figure}[ht!]
	\centering
	\includegraphics[width = 0.9\textwidth]{elchem}
	\caption{Электрическая схема реализации регулятора}
	\label{elchem}
\end{figure}

На представленной схеме выполняются равенства
\begin{align}
\begin{cases}
R_5 = R_7\\
R_6 = R_8\\
C_3 = C_4
\end{cases}
\end{align}

Рассчитаем значения элементов схемы.
\begin{align}
\begin{cases}
	\begin{cases}
		T_0 = R_1C_1 = 1\\
		T_2 = R_2C_1 = 0.5717
	\end{cases}\\
	\begin{cases}
		T_k = \frac{R_4}{C_2} = 4.4\\
		K_v+\sqrt{2} = \frac{R_4}{R_3} = 39.3452\\
	\end{cases}\\
	\begin{cases}
		\frac{R_6}{R_5} = 1\\
		T_4 = \frac{R_6}{C_3} = 0.1233\\
	\end{cases}
\end{cases}
\end{align}

При решении системы получаем следующие параметры:
\begin{gather}
\begin{cases}
	\begin{cases}
		R_2 = 1 \text{ Ом}\\
		C_1 = 0.5717 \text{ Ф}\\
		R_1 = 1.7491 \text{ Ом}
	\end{cases}\\
	\begin{cases}
		R_3 = 0.5 \text{ Ом}\\
		R_4 = 19.6726 \text{ Ом}\\
		C_2 = 4.4710 \text{ Ф}\\
	\end{cases}
	\begin{cases}
		R_5 = R_6 = R_7 = R_8 = 1.254 \text{ Ом}\\
		C_3 = C_4 = 10.1703 \text{ Ф}\\
	\end{cases}
\end{cases}
\end{gather}
