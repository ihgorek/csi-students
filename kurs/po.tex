\section{Анализ устойчивости неизменяемой части системы}
Использовав данные из таблицы \ref{initTab}, получим передаточную функцию неизменяемой части системы:
\begin{align}
W_H(s) = \frac{195}{(0.00324s^3 + 0.198s^2 + s)s}.
\end{align}


Схема моделирование представлена на рисунке \ref{Wnch}.
\begin{figure}[ht!]
	\centering
	\includegraphics[width = 0.85\textwidth]{Wnch}
	\caption{Схема моделирования замкнутой системы}
	\label{Wnch}
\end{figure}

При симуляции был получен график, приведённый на рисунке \ref{W_nchgraph}. По графику переходного процесса видно, что система неустойчива. Также это можно определить по корням функции.\par
\begin{figure}[ht!]
	\centering
	\includegraphics[width = 0.85\textwidth]{W_nch}
	\caption{Реакция замкнутой системы на единичное входное воздействие}
